# BrightnessCTL GUI
small **lightweight** GUI using **BrightnessCTL** for **wayland**, made in python. 

## Installation
*Note : you **MUST** have python3 and pip3 installed on your machine.*

### Clone the repository
git clone https://github.com/Anthoxoo/BrightnessGui.git

cd BrightnessGui

### Make the installer executable
chmod +x install.sh

### Run the installer
./install.sh

## Usage
chmod +x start.sh

Look for Brightness Control in your apps !
